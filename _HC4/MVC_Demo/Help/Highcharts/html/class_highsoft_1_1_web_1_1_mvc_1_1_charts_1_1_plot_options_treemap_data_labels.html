PlotOptionsTreemapDataLabelsAlign Highsoft.Web.Mvc.Charts.PlotOptionsTreemapDataLabels.Align
getset

The alignment of the data label compared to the point. If

right, the right side of the label should be touching the point. For points with an extent, like columns, the alignments also dictates how to align it inside the box, as given with the inside option. Can be one of "left", "center" or "right".

PlotOptionsTreemapDataLabelsOverflow Highsoft.Web.Mvc.Charts.PlotOptionsTreemapDataLabels.Overflow
getset

How to handle data labels that flow outside the plot area. The default is

justify, which aligns them inside the plot area. For columns and bars, this means it will be moved inside the bar. To display data labels outside the plot area, set crop to false and overflow to "none".
