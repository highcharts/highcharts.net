bool Highsoft.Web.Mvc.Stocks.ChartZooming.SingleTouch

Enables zooming by a single touch, in combination withchart.zooming.type. When enabled, two-fingerpinch will still work as set up by chart.zooming.pinchType. However, singleTouch will interferewith touch-dragging the chart to read the tooltip. And especiallywhen vertical zooming is enabled, it will make it hard to scrollvertically on the page.
