PlotOptionsColumnrangeStickyTracking Property
Sticky tracking of mouse events. When true, the
Copy
mouseOut
event on a series isn't triggered until the mouse moves over another series, or out of the plot area. When false, the
Copy
mouseOut
event on a series is triggered when the mouse leaves the area around the series' graph or markers. This also implies the tooltip. When
Copy
stickyTracking
is false and
Copy
tooltip.shared
is false, the tooltip will be hidden when moving the mouse between series. Defaults to true for line and area type series, but to false for columns, pies etc.
